販売所とは

運営会社が保有するコインをユーザーが提示された価格で売買する仕組み

取引所とは

ユーザー同士が仮想通貨を売買する

販売所と取引所の違い

取引所はユーザー同士のマッチングが上手くいかないと取引が成立しない

販売所は取引所より手数料がかかってしまうのがデメリット

初心者は取引所だと仮想通貨の値動きがどのように反映されているのか分かりづらいため、販売所で取引する方が安心ん

現物取引とは

手持ちの仮想通貨とほかの仮想通貨とを交換することを指す

取引手数料が掛かるが時価での販売が可能

資産額の範囲内でしか取引できないので取引によって資産を減らすリスクはあっても借金を負う心配がないのは、レバレッジ取引との違い

レバレッジ取引と違って値上がり時にしか利益を見込めない方法であるため、相場を見極めて値上がりのチャンスをうかがうのがいい

その一方でリスクを最小限に抑えて取引るのに最適

指値注文とは？

買う値段か売る値段のどちらかを指定して注文する方法

買い注文の場合は指値以下、売り注文の場合は指値以上の値段にならないと成立しない

成行注文とは？

売買の値段を指定せずに注文する方法

買い注文の場合はその時点の最低価格の売り注文に対応、売り注文の場合は最高額の買い注文に対応して注文が成立する

レバレッジ取引とは？

取引所に証拠金となる通貨を預けて仮想通貨を借り受けてそれを運用する方法

日本円を証拠金として買いと売りを両方行う事が出来る

現物取引は必ず買いから始めなければならないがレバレッジ取引は売りから始められる

仮想通貨が値下がり中でも利益が出る

売買数が極端に少なくなることで取引が成立しなくなり、売りたいときに売れなくなる流動性リスクが生じる

取引にかかる手数料のほかに借入手数料がかかり、利益が生じた場合に利益手数料がかかる

借りた期間内に運用した後は、借りた分を返さなければならない

出来高とは

取引が成立した金額の累計を指し、基本的にチャートの一番下に表示される

出来高が多いというのは投資家などがオーダーを活発に出し、取引が次々に成立していることを意味する

仮想通貨の税金について

仮想通貨で年間２０万円以上の所得があれば確定申告が必要となる

仮想通貨を売却（日本円に換金）した場合、その売却価格と取引価格との差が所得金額になる

120万円でビットコイン（BTC）を2BTC購入し、年内に0.5BTCを40万円で売却した場合の所得額は以下のとおりです。 40万円-（120万円÷2BTC）x 0.5BTC = 10万円 → 所得額は10万円

仮想通貨で商品を購入した場合、その取引時点での商品価格と仮想通貨の取引価格との差が所得金額になる

120万円でビットコイン（BTC）を2BTC購入し、年内に15万円の商品を0.2BTCで購入した場合の所得額は以下のとおりです。 15万円 -（120万円÷2BTC）x 0.2BTC = 30,000円 → 所得額は30,000円 また、100万円でビットコイン（BTC）を10BTC購入した後に、1BTC＝20万円に値上がりし、100万円（5BTC）で商品を買った場合、所得額は以下のようになります。 100万円 -（100万円÷10BTC）x 5BTC = 50万円 → 所得額は50万円

仮想通貨と仮想通貨の交換の場合、交換した仮想通貨のその時点での取得価格と保有していた仮想通貨の取得価格との差が所得金額になる

例えば、120万円でビットコイン（BTC）を2BTC購入し、年内にほかの仮想通貨を100万円分買うために1BTCを支払った場合の所得額は、以下のとおりになります。 100万円 -（120万円÷2BTC）x 1BTC = 40万円 → 所得額は40万円

確定申告における所得の計算方法は移動平均法と総平均法がある

移動平均法とは

仮想通貨を購入する度に計算する方法で購入額の合計を仮想通貨の数量で割ることで単価を算出する

そして売却時の価格から購入時の単価を差し引いた金額が所得となる

総平均法とは

一年間に取得した仮想通貨の取得価格の総額を、取得した仮想通貨の数量合計で割ることで算出する

計算が簡単だが誤差が生じやすい計算法

国税庁は移動平均法を推奨している

一度選択した計算方法は変更する事が出来ない

取引記録はこまめに保存を

仮想通貨の所得金額の算出には取引所の管理画面から取引履歴と確認しながら行いましょう

仮想通貨は雑所得に区分される

株やFXの先物取引によって損失が出た場合は、翌年以降3年間にわたって、損失の繰り越しが可能ですが、仮想通貨の損失は繰り越しができません。1年前に1,000万円の損失があったとしても、今年に入って1,000万円の利益が出た場合に課税される額は1,000万円となります

株式などは確定申告によって損益を相殺する事が出来るが仮想通貨はできない

雑所得は総合課税の対象となるため、給与所得などと合計しその金額によって所得税率が決まる

仮想通貨は累進課税の総合課税が適用なので1800万円超4000マ年以下の利益に対する累進課税率は40%となり4000万円を超える大きな収益を上げた場合には累進課税率は45%

今年の赤字を来年に繰り越せるか

仮想う通貨で生計を立てていて、事業所得であるという程度の規模であれば、赤字となった翌年に収益が上がった場合、確定申告をして赤字を相殺できる
